# Guide Writing Guide

How to write effective AI guides for this project.

## Format

- All guides MUST be titled bullet lists
- Start with clear title and one-line description
- One guide per topic

## Content

- Write rules, not explanations
- State what to do, not why (unless critical)
- Use CAPITAL keywords for constraints: NEVER, ALWAYS, MUST, PREFER

## Context

- AI picks up format and style from existing guides in this folder
- Match tone and structure of similar existing guides
